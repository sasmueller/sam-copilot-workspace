# SAM Copilot Workplace

A single workspace for customer strategy, business value assessments, MEDDPICC tracking, executive preparation, and portfolio management.

## Structure
- `accounts/` — account plans, stakeholder maps, and strategy notes
- `opportunities/` — opportunity briefs, pursuit plans, and deal support
- `meddpicc/` — qualification scorecards and deal inspection notes
- `exec-prep/` — executive briefing prep and meeting readouts
- `portfolio/` — portfolio reviews, priorities, and summaries
- `templates/` — reusable working templates
- `examples/` — sample filled-in documents
- `docs/` — workspace guidance

## Suggested workflow
1. Start with the account context in `accounts/`
2. Track active deals in `opportunities/`
3. Use `meddpicc/` for qualification rigor
4. Prepare leadership engagements in `exec-prep/`
5. Roll up status and themes in `portfolio/`
