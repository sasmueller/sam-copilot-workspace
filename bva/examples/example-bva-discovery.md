# Example BVA Discovery

This sample shows how an initial discovery worksheet could be structured before real customer data exists.

## Customer Context
- Account: Example Manufacturing Co.
- Industry: Manufacturing
- Business unit: Operations
- Initiative: Process optimization
- Sponsor: VP Operations

## Current State
- Current process: Manual reporting across plants
- Pain points: Delayed insights, inconsistent data, high admin effort
- Baseline metrics: Reporting cycle time, hours spent, error rate
- Constraints: Limited IT bandwidth
