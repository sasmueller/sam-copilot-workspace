# BVA Workspace

Use this area to manage business value assessments from hypothesis through customer-ready output.

## Suggested structure
- `templates/` — reusable BVA templates
- `examples/` — sample BVA artifacts
- `drafts/` — working assessments in progress
- `outputs/` — finalized customer-facing summaries

## Typical workflow
1. Capture customer context and initiative scope
2. Define baseline assumptions
3. Estimate value levers and impact ranges
4. Summarize risks, dependencies, and required inputs
5. Produce an executive-ready output
